# fire-atlas-app
UEA impact study app (sandbox for testing)

## Contents

* `ar6.land-no-ocean-BA_Total-yearly.html` - HTML script to generate an interactive IPCC AR6 land region world map for BA_Total (yearly)
* `ar6.land-no-ocean-BA_Total-monthly.html` - HTML script to generate an interactive IPCC AR6 land region world map for BA_Total (monthly)
* `ar6.land-no-ocean-BA_Forest_NonForest-yearly.html` - HTML script to generate an interactive IPCC AR6 land region world map for BA_Forest_NonForest (yearly)
* `ar6.land-no-ocean-BA_Forest_NonForest-monthly.html` - HTML script to generate an interactive IPCC AR6 land region world map for BA_Forest_NonForest (monthly)
* `ar6.land-no-ocean-Cem_Total-yearly.html` - HTML script to generate an interactive IPCC AR6 land region world map for Cem_Total (yearly)
* `ar6.land-no-ocean-Cem_Total-monthly.html` - HTML script to generate an interactive IPCC AR6 land region world map for Cem_Total (monthly)
* `ar6.land-no-ocean-Cem_Forest_NonForest-yearly.html` - HTML script to generate an interactive IPCC AR6 land region world map for Cem_Forest_NonForest (yearly)
* `ar6.land-no-ocean-Cem_Forest_NonForest-monthly.html` - HTML script to generate an interactive IPCC AR6 land region world map for Cem_Forest_NonForest (monthly)

The first step is to clone the latest fire-atlas-app repo and step into the check out directory: 

    $ git clone https://github.com/patternizer/fire-atlas-app.git
    $ cd fire-atlas-app

### Usage

The code was tested locally in a Python 3.8.16 virtual environment.

    $ firefox ar6.land-no-ocean-BA_Total-yearly.html (or any of the others as they're interlinked)
